# Extension Manager

Extension Manager 是一个用于 Chrome 扩展管理的扩展。

这个扩展管理器的核心定位是，通过规则定义，自动启用或禁用指定的浏览器扩展。

其它功能包括：

- 通过切换分组，手动批量的启用或禁用扩展
- 在 Popup 中临时手动启用或禁用扩展
- 支持为扩展添加别名和备注，以便快速搜索（扩展很多时比较有用）

- 上架 Edge

## 权限说明

本扩展在使用过程中，需要的浏览器权限。

- `读取您的浏览记录`

目的是读取打开的标签页的中的 URL, 根据当前 URL 来进行规则和匹配和执行

> 本插件管理工具不收集任何您的浏览记录，所有的规则判断都在本地运行

[chrome.tabs - Chrome Developers](https://developer.chrome.com/docs/extensions/reference/tabs/)

- `管理您的应用、扩展程序和主题背景`

用于获取安装的插件列表，并根据规则禁用或者启用指定的扩展

[chrome.management - Chrome Developers](https://developer.chrome.com/docs/extensions/reference/management/)
